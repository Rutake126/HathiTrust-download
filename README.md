# HathiTrust 图片下载脚本

基于 **DrissionPage** 库开发的自动化下载工具，模拟真实浏览器行为，实现 HathiTrust 图书页面图片的自动化保存。

## 使用须知

脚本目前还有很多没有完善的地方。

##  使用说明

1. **环境准备**：确保已安装 `DrissionPage` 库。
2. **配置代理**：在代码中正确填写你的代理地址（如 `127.0.0.1:7897`）。
3. **运行脚本**：执行 `main.py` 即可开始下载。
