# HathiTrust 图片下载脚本

基于 **DrissionPage** 库开发的自动化下载工具，模拟真实浏览器行为，实现 HathiTrust 图书页面图片的自动化保存。

## 使用须知

- 本工具仅供学术研究与个人学习使用，请遵守 HathiTrust 的使用条款。
- 下载频率过高可能触发服务器限流，建议合理设置下载间隔。

## 功能特性

| 功能 | 说明 |
|------|------|
| URL 智能解析 | 支持阅读器链接、图像链接、纯 Book ID 三种输入方式 |
| 下载质量 | 300 PPI（默认）/ 600 PPI（高质量） |
| 批量下载 | 自定义页码范围，支持 1-10000 |
| 代理支持 | 可开启/关闭，自定义代理地址 |
| 暂停/继续 | 下载过程中随时暂停，恢复后继续 |
| 自动重试 | 失败后按配置次数和间隔自动重试 |
| 自动归档 | 以书籍 ID 创建子文件夹，页面图片保存其中 |
| 实时日志 | 深色终端风格日志，显示下载进度与统计 |

## 使用说明

### 环境准备

```bash
pip install DrissionPage customtkinter
```

### 方式一：GUI 图形界面（推荐）

```bash
python gui.py
```

操作流程：
1. 设置代理（可选，默认 `127.0.0.1:7897`）
2. 粘贴 HathiTrust 链接，点击「解析」提取 Book ID
3. 设置页码范围、下载质量、下载间隔
4. 点击「开始下载」

### 方式二：命令行交互

```bash
python main.py
```

按提示依次输入代理设置、链接、质量、页码范围即可。

## 支持的链接格式

```
# 阅读器页面链接
https://babel.hathitrust.org/cgi/pt?id=keio.10812543530&seq=2

# 图像服务器链接
https://babel.hathitrust.org/cgi/imgsrv/image?id=keio.10812543530&attachment=1&format=image%2Fjpeg&size=ppi%3A300&seq=1

# 纯 Book ID
keio.10812543530
```
